##这是一个基于jquery的摸态框插件可以自由定制需要的样式操作
可以对样式进行自我定制

主要参数有：
width:400,	//弹框宽度
height:"",	//弹框高度（不传则则根据内部内容自适应）
title:"提示",	//弹框标题
content:"",	//显示内容（可以传递html片段作为参数，并可以自定义内部的样式）
okBtn:"",	//成功按钮（不传则隐藏改按钮）
cancleBtn:"",	//取消按钮（不传则隐藏改按钮）
okCallback:"",	//成功按钮回调函数，传递该参数需要自己关闭弹窗；
cancleCallback:"",//取消按钮回调函数，传递该参数需要自己关闭弹窗；
closeCallback:""//关闭按钮回调函数，传递该参数需要自己关闭弹窗；
